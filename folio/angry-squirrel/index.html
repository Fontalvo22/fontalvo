ERC721A

FEATURES

The angry squirrel club smart contract will allow multiple NFTs to be minted for the same cost as minting a single NFT.

ERC721A is an implementation of regular ERC721, that allows gas savings for minting multiple NFTs in a single transaction.

We have audited and partnered with a external security consultant to ensure our contract is solid and secure.

CONTEXT

One of the biggest issue on Ethereum blockchain are the Gas fees. NFTs developers on the Ethereum blockchain need to adapt with new technologies to reduce gas fees that have been way too high for long time.

The Angry Squirrel Club developer team wanted to optimize this smart contract for allow multiple NFTs to be minted for a gas fee equivalent to minting a single NFT.
